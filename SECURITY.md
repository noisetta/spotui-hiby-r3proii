# Security policy

This repository should never contain private credentials, tokens, WiFi passwords, Spotify cache files, firmware images, or device-specific backups.

Please do not open public issues containing:

- Spotify credentials or cache files
- WiFi SSIDs/passwords
- access tokens or refresh tokens
- private logs containing account information
- proprietary firmware images or extracted proprietary binaries

If you accidentally committed or shared a secret, rotate that credential immediately. Deleting it from a commit or issue may not fully remove it from history.

For sensitive reports, contact the maintainer privately rather than posting secrets publicly.
